ESTUDO PARA A PROVA

Codificação: regra que diz exatamente quantos bytes um caracter especial possui.

HTML: Linguagem descritiva, estrutura e semântica de um texto. Uso de marcação de texto.

Unicode: UTF-8, fornecer conteúdo em qualquer linguagem humana existente. Codificação mínima necessária que um navegador deve ter suporte.

Elementos em bloco: "div".
Elementos em linha: "span".

Main: conteúdo principal de um documento

"Section": partes de um outro elemento no qual o conteúdo é relacionado.

"BR /": quebra de linha.

"strong": grifo.
"em": ênfase.
"i": itálico.
"mark": marca-texto.

"link": Âncora na head, definir relações do documento com recursos externos.
"href": define url do recurso alvo.
"rel": defie relação do recurso com o documento atual.

"a": Âncora no body.
"download": define se o recurso é pra ser baixado diretamente.
"target": especifica onde o recurso deve ser aberto.

Listas:
- Listas não ordenadas:
"ul", ordem não é uma informação importante, cada item da lista corresponde a um "li".
- Listas ordenadas:
"ol", a ordem importa dentro do contexto e parte integrante da informação passada,"type=": símbolo de ordenação. Cada item da lista corresponde a um "li".
- Listas associativas:
"dl", termos seguidos de descrições. "dt": termo. "dd": definição do termo.

Tabelas: "Table". Organizar dados em linhas e colunas.
"tr": divide os dados em linhas.
"td": divide os dados em colunas.
"td": divide os dados das linhas em colunas, organização em estrutura de árvore. "th": marca a célula como título para um grupo de dados. Sendo o grupo de dados especificado por "scope=" ("th scope=").
O elemento "caption" dá um título ou legenda para a tabela. Deve ser o primeiro elemento após "table".
"thead" dá um cabeçalho para a tabela. Deve aparecer após "caption".
"tfoot" define um rodapé para a tabela, independente onde estiver, vai acrescentar a linha por último.
"tbody" define um corpo de dados para a tabela. Deve vir após "thead" e "tfoot".

CSS:
Chave de propriedade: nome chave que representa uma característica de exibição.
Valor da propriedade: valor associado a chave que tem efeito na exibição.
Seletores: especifica quai elementos serão alvos das regras.

Para relacionar estilos através de link externo, deve-se usar (link rel="stylesheet" href="arquivo.css").
Seletor universal: "*".

RESUMO:
CSS EM LINHA: (elemento style="atributo")
CSS EMBUTIDO: (style) elemento { atributo } (/style).
CSS EXTERNO: (link rel="stylesheet" href="arquivo.css").
